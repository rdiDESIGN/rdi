##一个展示的页面.

作为图形设计师，如今不得不跨界去尝试学习html，对，仅仅是html，很初级的一些页面的学习制作。

然而计算机专业的工程师，又对我们的设计作品感兴趣，很多矢量图被转换成SVG用于丰富页面。
遗憾的是，网站设计转图形设计很容易，而图形设计，转网页设计很难。设计是互通的，但目前感觉，网站设计对平面图形设计师是降维按地上磨擦。
svg都学了好久，希望还有头发去折腾。

在此感谢作者将代码分享出来可以供个人或者商业用途。
